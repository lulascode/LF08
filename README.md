# FPR - Fantasy Pig Racing

## Ziel:
- Web-App mit integriertem Youtube Video
- Datenbank mit definierten Race Start/End-Zeiten
- Timer bis zum Nächsten Race
- Tracking über die Wetten
- "Random Race" - Random Video mit Random Race (via Timestamp)
